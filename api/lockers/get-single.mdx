---
title: "Get Single Locker"
api: "GET /api/lockers/{lockerNo}"
description: "Retrieve a specific locker by its locker number"
---

## HTTP Method and Path

```
GET /api/lockers/{lockerNo}
```

## Path Parameters

<ParamField path="lockerNo" type="string" required>
  The unique locker number to retrieve
</ParamField>

## Response

Returns a single locker object.

<ResponseField name="lockerNo" type="string" required>
  The unique locker number identifier
</ResponseField>

<ResponseField name="employeeNumber" type="string">
  The employee number assigned to this locker (null or empty if unassigned)
</ResponseField>

<ResponseField name="size" type="integer" required>
  The size of the locker
</ResponseField>

<ResponseField name="location" type="string" required>
  The physical location of the locker
</ResponseField>

<ResponseField name="isEmpty" type="boolean" required>
  Indicates whether the locker is currently empty
</ResponseField>

## Example Request

```bash cURL
curl -X GET https://api.example.com/api/lockers/A001 \
  -H "Content-Type: application/json"
```

## Example Response

### Success (200 OK)

```json
{
  "lockerNo": "A001",
  "employeeNumber": "EMP12345",
  "size": 2,
  "location": "Building A - Floor 1",
  "isEmpty": false
}
```

### Error (404 Not Found)

```json
"Locker not found"
```
